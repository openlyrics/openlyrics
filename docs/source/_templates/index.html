Welcome

OpenLyrics is a free, open XML standard for Christian worship songs. The goal of OpenLyrics is to provide an application-independant and operating system-independant song format for interoperability between applications.

The following features can be seen in OpenLyrics:

  • XML format
  • Multiple languages
  • Chords
  • Transliteration
  • XML schema
  • and more...

Examples

Song examples can be found on the OpenLyrics data format page. Links to some implementations of OpenLyrics can be found on the Projects using OpenLyrics page.

Get OpenLyrics

OpenLyrics is available as an zip archive containing a few song examples, XML schema and a simple script for validation.

The latest development version can be found in a Subversion repository, at https://changingsong.svn.sf.net/svnroot/changingsong/trunk/openlyrics

The Subversion repository can be browsed online.
