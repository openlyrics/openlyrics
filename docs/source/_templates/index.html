Welcome

OpenLyrics is a free, open XML standard for Christian worship songs. The goal of OpenLyrics is to provide an application-independant and operating system-independant song format for interoperability between applications.

The following features can be seen in OpenLyrics:

  • XML format
  • Multiple languages
  • Chords
  • Transliteration
  • XML schema
  • and more...

Examples

Documentation on the song format description can be found on the OpenLyrics data format page and song examples in the available zip archive.

Implementations

Projects that support OpenLyrics and also Libraries for a number of programming languages can be found on the Projects using OpenLyrics page.

Get OpenLyrics

OpenLyrics is available as an zip archive containing some song examples, XML schema and and a RELAX NG schema for validation.

The latest development version can be found in the Github repository.

Get a local copy of the repository with this command:

git clone https://github.com/openlyrics/openlyrics.git
